# Pet Memorial SaaS

Base inicial da aplicação para abrir no VSCode e começar o desenvolvimento.

## Requisitos
- Node.js 20+
- npm 10+

## Como iniciar
```bash
npm install
npm run dev
```

## Estrutura inicial
- `src/app`: rotas e layouts
- `src/components`: componentes compartilhados
- `src/lib`: integrações e utilitários
- `public`: arquivos públicos
